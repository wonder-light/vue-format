# vue-format 扩展

用于vue文件的格式化扩展

## 说明

为Vue文件提供一个格式化命令
可以格式化\<template>，\<script>,\<style>中的内容
可以单独配置 html, javascript, css 功能

## 使用

在文本编辑器中，打开右键菜单，选择“格式化vue”即可格式化文件
